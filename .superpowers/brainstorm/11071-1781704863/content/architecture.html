System Architecture

Four layers — each with one clear job

UI Layer

PyQt6 Window + System Tray

Main window (folder picker, schedule config, retention, password, logs, progress bar) + tray icon with right-click menu (Backup Now, Open, Quit)

↕
Backup Engine

backup_engine.py

Zips the two selected folders → AES-encrypts the archive → uploads to IONOS → applies retention policy (deletes old archives beyond limit)

↕
Scheduler

scheduler.py

Runs in a background thread. Triggers the backup engine on the configured interval (hourly / daily / weekly + time-of-day)

Config Store

config.json

Stores folder paths, IONOS credentials (bucket, endpoint, access key, secret), schedule, retention count, encrypted password hash

↕
IONOS Object Storage

boto3 (S3-compatible)

Uploads encrypted .zip archives. Each file named backup-YYYY-MM-DD-HH-MM.zip.enc. Bucket listing used for retention cleanup.

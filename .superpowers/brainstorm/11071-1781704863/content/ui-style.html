What kind of Windows app do you want?

Choose the UI style that best fits how you'll use this backup tool

A

System Tray App

Runs silently in the background. Right-click tray icon to open settings, trigger backup, or see status. Minimal, always-on feel. Best for "set it and forget it" use.

B

Desktop Window App

A full window you open when you need it. Shows backup history, folder selection, schedule settings, and progress in one place. Best if you want full visibility and control.

C

Both: Tray + Window

Tray icon for quick status and triggering. Click to open the full window for configuration and logs. The most complete experience — more to build, but most polished result.
